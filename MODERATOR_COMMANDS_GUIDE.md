# 👮 دليل أوامر إدارة الروم والمشرفين

## أوامر الترقية والإزالة

### 1. 👮 إضافة مشرف
**الأمر:**
```
addmod @username
```

**الوصف:** ترقية مستخدم ليصبح مشرف في الروم
**الصلاحيات:** للأدمن والمشرفين فقط
**مثال:**
```
addmod @ahmed
```
**النتيجة:** 
```
✅ تم ترقية @ahmed كمشرف! 👮
```

---

### 2. 🎨 إضافة مصمم
**الأمر:**
```
adddesigner @username
```

**الوصف:** ترقية مستخدم ليصبح مصمم الروم
**الصلاحيات:** للأدمن والمشرفين فقط
**مثال:**
```
adddesigner @fatima
```
**النتيجة:**
```
✅ تم ترقية @fatima كمصمم! 🎨
```

---

### 3. 🗑️ إزالة صلاحيات المشرف
**الأمر:**
```
removemod @username
```

**الوصف:** إزالة صلاحيات المشرف من مستخدم
**الصلاحيات:** للأدمن والمشرفين فقط
**مثال:**
```
removemod @ahmed
```
**النتيجة:**
```
✅ تم إزالة صلاحيات @ahmed كمشرف
```

---

### 4. 🗑️ إزالة صلاحيات المصمم
**الأمر:**
```

```

**الوصف:** إزالة صلاحيات المصمم من مستخدم
**الصلاحيات:** للأدمن والمشرفين فقط
**مثال:**
```
removedesigner @fatima
```
**النتيجة:**
```
✅ تم إزالة صلاحيات @fatima كمصمم
```

## 🎯 حالات الاستخدام الشائعة:

### تنظيم الروم
```
// ترقية شخص نشط كمشرف
addmod @active_user

// ترقية مصمم جديد للروم
adddesigner @designer_name

// إزالة صلاحيات مشرف غير فعال
removemod @inactive_mod

// إزالة مصمم وترقية آخر
removedesigner @old_designer
adddesigner @new_designer
```

---

## ⚠️ ملاحظات مهمة:

✅ **متطلبات:**
- المستخدم يجب أن يكون موجود في الروم الحالي
- يجب أن تكون لديك صلاحيات مشرف أو أعلى
- استخدم @ قبل الاسم (اختياري - البوت يحتمل مع وبدون)

❌ **ممنوع:**
- لا تقدر ترقي مالك الروم (owner)
- لا تقدر ترقي شخص أعلى منك مستوى

💡 **نصائح:**
- تأكد من كتابة الاسم صح
- الأوامر حساسة لحالة الأحرف (lowercase/uppercase)
- البوت بيرد عليك إذا حصلت مشكلة

---

## 📋 الهرمية:

```
Owner (مالك الروم)
    ↓
Moderator (مشرف)
    ↓
Designer (مصمم)
    ↓
Normal User (مستخدم عادي)
```

---

## 🔍 معلومات إضافية:

### المشرف (Moderator):
- يمكنه طرد (kick) المستخدمين
- يمكنه كتم (mute) المستخدمين
- يمكنه حظر (ban) المستخدمين
- يمكنه إدارة الروم

### المصمم (Designer):
- يمكنه تصميم الروم
- يمكنه تغيير الأثاث
- يمكنه تعديل الإضاءة والألوان
- يمكنه إدارة مظهر الروم

---

## 🆘 حل المشاكل:

**المشكلة:** رسالة "لا تملك صلاحية"
- **الحل:** تأكد أنك مشرف أو صاحب الروم

**المشكلة:** رسالة "غير موجود في الروم"
- **الحل:** تأكد أن المستخدم موجود في الروم حالياً

**المشكلة:** رسالة "حصلت مشكلة"
- **الحل:** جرب مرة أخرى أو اتصل بالدعم
