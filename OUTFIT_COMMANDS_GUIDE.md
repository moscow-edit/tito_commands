# 👗 دليل أوامر اللبس (Outfit Commands)

## أوامر اللبس المتاحة:

### 1. 📋 `outfit @username`
**الوصف:** عرض ملابس أي مستخدم في الروم أو خارجه
**مثال:**
```
outfit @ahmed
```
**النتيجة:** يعرض قائمة جميع الملابس التي يرتديها أحمد

---

### 2. 👕 `equip @username`
**الوصف:** نسخ ملابس مستخدم آخر بالضبط
**الصلاحيات:** للأدمن والمشرفين فقط
**مثال:**
```
equip @fancy_user
```
**النتيجة:** البوت يرتدي نفس ملابس المستخدم fancy_user

---

### 3. 🎨 `equip [preset_name]`
**الوصف:** ارتداء لبس محفوظ مسبقاً
**الخيارات المتاحة:**
- `equip goth` - لبس قوطي
- `equip angel` - لبس ملاك
- تركيبات أخرى حسب الملابس المحفوظة

**مثال:**
```
equip angel
```

---

### 4. 🗑️ `delequip`
**الوصف:** حذف/إزالة اللبس الحالي للبوت
**الصلاحيات:** للأدمن فقط
**مثال:**
```
delequip
```

---

### 5. 🔄 `changeoutfit`
**الوصف:** العودة للملابس الافتراضية/الأساسية
**الصلاحيات:** للأدمن فقط
**مثال:**
```
changeoutfit
```

---

## كيفية حفظ لبس جديد:

### خطوة 1: البحث عن أجمل لبس
انظر حول الروم واختر شخصاً به ملابس جميلة

### خطوة 2: نسخ اللبس
```
equip @اسم_المستخدم
```

### خطوة 3: حفظ اللبس (للأدمن)
```
savefit تسمية_اللبس
```

---

## 💡 نصائح مهمة:

✅ **يمكن للجميع:** عرض ملابس الآخرين بأمر `outfit`
❌ **فقط الأدمن:** تغيير ملابس البوت (`equip`, `delequip`, `changeoutfit`)
🎨 **اللبس المحفوظ:** يبقى محفوظاً حتى لو أعاد البوت التشغيل

---

## أمثلة استخدام:

```
// عرض ملابس أحمد
outfit @ahmed

// البوت يرتدي نفس ملابس فاطمة
equip @fatima

// البوت يرتدي لبس القوط المحفوظ
equip goth

// إزالة جميع الملابس
delequip

// العودة للملابس الأساسية
changeoutfit
```

---

## 📞 للمزيد من المساعدة:
اكتب `commands` أو `اوامر` في الهمس لتحصل على قائمة كاملة بجميع الأوامر!
